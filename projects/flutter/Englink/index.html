EngLink - All in one

Project Description

This project involved developing an innovative app that connects car owners with repair stations for emergency support, providing real-time assistance on the map. Features include direct communication with engineers for consultations, an admin dashboard for managing and adding repair stations, engineers, and customers, and real-time tracking and notification systems.

Technologies Used

  • Flutter
  • Dart
  • Firebase
  • Google Maps API
  • REST API
  • Figma
